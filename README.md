* Responsive modern UI
* Interactive animations
* About section
* Skills showcase
* Project gallery
* Contact section
* Clean and professional design

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/Kirtivardhans123/My-Portfolio.git
```

Open project folder:

```bash
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
